# Neovim config (vim.pack)

This config now uses Neovim built-in package manager `vim.pack` (Neovim >= 0.12).

## Plugin management

- Plugin source list: `lua/config/pack.lua`
- Plugin setup entry: `lua/plugins/init.lua`
- Lockfile (generated by Neovim): `~/.config/nvim/nvim-pack-lock.json`

## Loading strategy

- **Eager on startup**: core config modules and most plugin setup modules in `lua/plugins/init.lua`
- **On-demand via keymaps/commands**: selected plugins use `require("config.pack").load(...)` before setup/usage
  - `telescope.nvim`
  - `neo-tree.nvim`
  - `trouble.nvim`
  - `persistence.nvim` (now only loaded when restoring session)
  - `vim-translator`
  - `copilot.vim` (gated by `NVIM_ENABLE_COPILOT=1`)

## Recent cleanup

- Removed duplicated mini specs (`mini.ai` / `mini.icons` / `mini.pairs`) and kept `mini.nvim` as the single source.
- Trimmed Catppuccin integrations to currently installed/used plugins.
- Removed archived commented module `lua/plugins/fzflua.lua`.

Useful commands:

- Update plugins: `:lua vim.pack.update()`
- Inspect installed plugins: `:lua vim.print(vim.pack.get())`
